# Advanced Barcode Printer - Complete Documentation

## Overview

The Advanced Barcode Printer provides complete control over barcode layout, sizing, and print settings. It includes:

- **5 Presets** for common layouts
- **Full Control**: Pages, size, scale, margins, gaps
- **Live Preview**: Visual page layout simulation
- **Batch Printing**: Print unlimited barcodes across multiple pages
- **Settings Export**: Save and share configurations

---

## Features

### 1. **Presets** (Pre-configured Templates)

Quick-start templates for common printing scenarios:

#### **2 Column - 50×25mm (Standard)** ⭐
- **Best for**: Standard label sheets
- **Layout**: 2 columns × 10 rows = 20 per page
- **Product**: Safa, Talwar Belts, Brooches, etc.
- **Pages for 40 items**: 2 pages
- **Use Case**: Most versatile, works with standard label sheets

#### **3 Column - 40×20mm**
- **Best for**: Space-efficient printing
- **Layout**: 3 columns × 14 rows = 42 per page
- **Product**: Small items, compact labels
- **Pages for 40 items**: 1 page
- **Use Case**: Save paper, compact layout

#### **4 Column - 30×20mm**
- **Best for**: Maximum density
- **Layout**: 4 columns × 13 rows = 52 per page
- **Product**: Tiny items, price tags
- **Pages for 40 items**: Less than 1 page
- **Use Case**: High-volume printing

#### **1 Column - 100×30mm (Large)**
- **Best for**: Large, readable barcodes
- **Layout**: 1 column × 9 rows = 9 per page
- **Product**: Premium items, marketing
- **Pages for 40 items**: 5 pages
- **Use Case**: Easy-to-scan, prominent display

#### **Thermal - 4×6 inch (Shipping)**
- **Best for**: Shipping labels
- **Layout**: 1 column × thermal paper
- **Product**: Logistics, parcels
- **Pages for 40 items**: 40 pages
- **Use Case**: Thermal printer labels

---

### 4. **Layout Settings**

#### Columns (1-6)
- **1 column**: Single-file printing
- **2 columns**: Balanced, standard
- **3+ columns**: Dense, space-saving

#### Margins (mm)
- **Top/Bottom**: 5-20mm
- **Left/Right**: 5-20mm
- **Purpose**: Printer bleed area

#### Gaps (mm)
- **Horizontal Gap**: 0-10mm (between columns)
- **Vertical Gap**: 0-5mm (between rows)
- **Purpose**: Spacing between barcodes

#### Scale (50% - 200%)
- **50%**: Half-size barcodes
- **100%**: Original size
- **150%**: Larger, more readable
- **200%**: Maximum size

#### Quality Options:
- **Low**: Fast, less clear
- **Medium**: Balanced
- **High**: Clearest, best for scanning

## Usage Guide

### Step 1: Open Advanced Printer
1. Go to **Inventory** page
2. Click any product → **View Product**
3. Click **Advanced Settings** button

### Step 2: Choose Your Preset (or Customize)

#### Option A: Use Preset
- Click any preset button
- Settings auto-apply
- Jump to Step 4

#### Option B: Custom Settings
- Modify **Paper Size**
- Set **Barcode Dimensions**
- Adjust **Margins & Gaps**
- Fine-tune **Scale**

### Step 3: Add Barcodes

#### Add Items:
1. Click **Add** button
2. Enter barcode code
3. Enter product name
4. Repeat for all items

#### Edit Items:
- Click on code/name to edit
- Delete items with trash icon

### Step 4: Preview

#### Check Calculations:
- See column/row counts
- Verify barcodes per page
- Check pages needed

#### Visual Preview:
- See scaled page layout
- Blue boxes = barcode positions
- Red boxes = margins
- Adjust if needed

### Step 5: Print

Click **Print Now**:
1. Browser opens print dialog
2. Verify printer selected
3. Check paper size
4. Disable headers/footers
5. Click **Print**

## Advanced Tips

### Tip 1: Finding Perfect Fit
- Use **Preview** to visualize
- Adjust **Scale** slider
- Watch **Barcodes Per Page** update
- Stop when satisfied

### Tip 2: Label Sheet Compatibility
```
Standard A4 Sheets:
├─ 2col × 50×25mm: Avery 5340
├─ 3col × 40×20mm: Generic A4
└─ 4col × 30×20mm: Generic A4

Thermal:
└─ 4×6 inch: Zebra/Godex labels
```

### Tip 3: Printer Calibration
- Print test page with 1 barcode
- Compare with label sheet
- Adjust **Margins** if misaligned
- Adjust **Scale** if size wrong
- Re-print until perfect

### Tip 4: Batch Efficiency
- Add all barcodes at once
- Use same preset for multiple batches
- Print all 40 at once = 2 pages

### Tip 5: Export Settings
- Click **Copy Settings** for batch printing
- Save to text file
- Reuse for future orders

## Troubleshooting

### Issue: Barcodes too small
**Solution**: 
- Increase **Scale** (move slider right)
- Increase **Barcode Width/Height**
- Choose preset with larger dimensions

### Issue: Barcodes too large
**Solution**:
- Decrease **Scale**
- Decrease dimensions
- Increase columns or gaps

### Issue: Not enough per page
**Solution**:
- Reduce margins
- Reduce barcode size
- Increase columns
- Decrease gaps

### Issue: Print alignment wrong
**Solution**:
- Check preview
- Adjust **Top/Left Margins**
- Print test page
- Calibrate printer

### Issue: Text not readable
**Solution**:
- Increase **Scale**
- Set Quality to **High**
- Increase barcode dimensions
- Use fewer columns

### Issue: Barcode won't scan
**Solution**:
- Increase **Barcode Width** (minimum 40mm)
- Set Quality to **High**
- Increase **Scale**
- Test with different scanner
